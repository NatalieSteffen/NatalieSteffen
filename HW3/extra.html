WURFL Capabilities

Capability Use
is_wireless_device allows developers to know if the connected device has wireless capabilities, specifically a phone or a tablet rather than a laptop or desktop PC. This can be very relevant to determine what kind of layout to show, as typically such devices are touch-enabled, which likely makes a different layout more intuitive than someone using a mouse or a trackpad.
max_image_width returns the numerical value of the widest image a user's decide can display in CSS pixels. This would be very useful for determining breakpoints if a webpage is image heavy.
form_factor returns the user's device category, such as Smart-TV, smartphone, desktop, etc. Knowing what type of device your users use to access your site can and should influence how you design future updates, and potentially redesign current content to better meet more user's needs.
